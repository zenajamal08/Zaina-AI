# Zaina-AI
في هذا المشروع طورت نموذج تعلم آلي لتصنيف الارقام المكتوبة بخط اليد  باستخدام خوارزميات خاضعة للإشراف. بدأت مراحل العمل بتحميل مجموعة البيانات وفحصها للتأكد من سلامتها وخلوها من القيم المفقودة، ثم تنفيذ خطوات تنظيف ومعالجة البيانات مثل تحويل القيم إلى صيغة رقمية وتقييس الخصائص باستخدام StandardScaler لتحسين أداء النموذج.

بعد ذلك تم تقسيم البيانات إلى بيانات تدريب واختبار، وتجربة أكثر من خوارزمية مثل الانحدار اللوجستي و SVM، ثم تقييم الأداء باستخدام مقاييس مثل الدقة (Accuracy)، وتقرير التصنيف (Classification Report)، ومصفوفة الالتباس (Confusion Matrix). كما تم تحليل النتائج ومقارنة أداء النماذج لاختيار النموذج الأفضل بناءً على الدقة والاستقرار وتقليل التحيز.

يهدف المشروع إلى بناء نموذج دقيق وقابل للتطبيق عملياً مع توثيق جميع مراحل التطوير والتحسين.
